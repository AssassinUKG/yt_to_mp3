# Youtube to mp3

## Requirments 

- Have ffmpeg installed. 
Windows 
```
choco install ffmpeg
```

## Usage

yt_to_mp3.py YoutubeLink_or_ID

```
./yt_to_mp3.py https://www.youtube.com/watch?v=dQw4w9WgXcQ
```

* Rick Astley youtube link above.
